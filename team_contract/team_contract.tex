With signatures from all members of Apex Automation, the team guarantees to deliver demonstrate a prototype with the following features at the final presentation:

\begin{itemize}
    \item Dispensing of different pills from designated pill containers using designed dispensing  mechanism at scheduled dispensing times/intervals.
    \item Creating pill dispensing schedules using designed online portal that will upload said schedule to the device.
    \item Device will emit audible reminders when pill(s) have not been retrieved after a set amount of time that will be determined at a later date.
    \item Interactive device UI that will be displayed on onboard device LCD. This UI will display device information, settings and progress updates during the dispensing process. The UI will be interacted with onboard device buttons.
    \item Collect the appropriate sensor data in order to keep adherence statistics and statistics about the device. This data should be accessible to the user through the online portal. 
\end{itemize}
